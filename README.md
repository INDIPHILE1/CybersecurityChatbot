# Cybersecurity Awareness Bot

A C# console chatbot that educates users about cybersecurity best practices.  
**Developed for PROG6221 assignment using .NET 8.0.**

## Features
- 🎤 Voice greeting (plays `Greeting.wav` at startup)
- 🎨 ASCII art cybersecurity logo
- 👤 Personalized user interaction (asks for name)
- 💬 Responds to cybersecurity questions (passwords, phishing, safe browsing, 2FA, malware)
- ✅ Input validation (handles empty or unknown inputs)
- 🎨 Enhanced console UI (colors, borders, typing effect)
- 📁 Clean code structure (separate classes)
- 🐙 GitHub with 6+ commits
- ⚙️ GitHub Actions CI (builds on every push)

## Requirements
- .NET 8.0 SDK
- Visual Studio 2022 (or any C# IDE)
- Git & GitHub account
